Redis的备份与恢复
2019-09-04|Redis入门

  • 备份
    dump.rdb：RDB方式的备份文件
    appendonly.aof：AOF方式的备份文件

  • 恢复

如果需要恢复数据，停掉 redis 将备份文件 (dump.rdb) 或（appendonly.aof）移动到 redis 安装目录并启动服务即可。
